Documentation de l'administration - Ilma Event

Cette section vous permet de gérer les différents éléments utilisés dans les formulaires et les listes déroulantes sur le site web public. Vous pourrez ajouter de nouvelles prestations, types d'événements et bien plus encore pour personnaliser l'expérience des utilisateurs.

› Ajouter un nouvel élément.
  1. Cliquez sur le bouton . Un popup s'ouvrira, vous permettant de saisir les détails de l'élément.
  2. Remplissez le champ "Libellé" qui est obligatoire. Vous pouvez également fournir une description, bien que celle-ci ne soit pas obligatoire et ne doit pas dépasser 250 caractères. Un compteur de caractères est disponible pour vous aider à respecter cette limite.
  3. Cochez la case "Actif" si vous souhaitez que l'élément soit visible dans les listes déroulantes ou les emplacements spécifiés sur le site. Par défaut, cette case est cochée.
  4. Saisissez un ordre si vous souhaitez spécifier la position de l'élément dans la liste. Ce champ n'est pas obligatoire.
  5. Pour les prestations, vous avez la possibilité d'ajouter une icône qui sera affichée avec la prestation sur la page d'accueil. Il est recommandé de choisir une icône représentative. Ce champ est obligatoire.
  6. Une fois que vous avez rempli les champs nécessaires, cliquez sur pour enregistrer l'élément. Il devrait maintenant apparaître dans la liste.

Pour trouver une icône appropriée, vous pouvez utiliser la bibliothèque d'icônes Font Awesome disponible sur fontawesome.com. Recherchez un mot-clé qui correspond à votre prestation, puis copiez la classe de l'icône au format HTML et collez-la dans le champ dédié.

Exemple HTML : <i class="fa-solid fa-user"></i>
Classe à copier : fa-solid fa-user

› Modifier un élément existant.
  1. Accédez à la liste contenant l'élément que vous souhaitez modifier.
  2. Pointez sur l'élément spécifique que vous souhaitez modifier, puis cliquez sur l'icône du stylet dans la colonne avant-dernière.
  3. Une fois cliqué, la même fenêtre contextuelle d'ajout s'ouvrira, mais cette fois elle sera pré-remplie avec les détails de l'élément sélectionné.
  4. Apportez les modifications nécessaires aux détails de l'élément.
  5. Enregistrez vos modifications en cliquant sur le bouton .

› Supprimer un élément existant.
  1. Accédez à la liste contenant l'élément que vous souhaitez supprimer.
  2. Cliquez sur l'icône de la corbeille dans la dernière colonne, en regard de l'élément que vous souhaitez supprimer.
  3. Une alerte s'ouvrira pour confirmer votre décision de suppression.
  4. Si vous êtes sûr de vouloir supprimer l'élément, cliquez sur . L'élément sera alors supprimé définitivement.
  5. Si vous changez d'avis, cliquez sur pour annuler la suppression.

En suivant ces instructions, vous pourrez facilement ajouter, modifier ou supprimer des éléments de votre site web, vous donnant ainsi un contrôle total sur son contenu et son fonctionnement et une expérience utilisateur plus riche et personnalisée.
